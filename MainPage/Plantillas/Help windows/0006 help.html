Ayuda sobre infotipo 6.

Dentro de este apartado se encuentra toda la informacion relacionada a las direcciones del empleado, tanto fiscal como primaria. Usualmente no hay tantos errores dentro de este infotipo.

Dentro del infotipo se puede encontrar:

  • Calle y numero
  • Colonia
  • Estado
  • Pais
  • Telefono de contacto
